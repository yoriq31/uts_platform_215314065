Tentang Saya

Perkenalkan nama saya Yoriqh Rizky Pratama Siman. Asal saya dari sebuah desa kecil Sorowako, Kabupaten Luwu Timur, Provinsi Sulawesi Selatan Saya senang kalian dapat mengenal saya dan saya pun senang mengenal kalian. Saat ini saya sedang menempuh perkuliahan Informatika di Universitas Sanata Dharma tepatnya Kampus III Paingan.

Hobi saya yakni bermain alat musik yaitu gitar, kemudian saya juga suka berenang, bermain badminton, dan bermain game. Terkadang saya juga dikala senggang biasa menghabisakan waktu dengan memasak. Skill memasak saya dapatkan dari mama saya yang juga suka memasak dan membuat beragam kue dan roti. Saat ini saya juga menjadi ketua sebuah organisasi.
